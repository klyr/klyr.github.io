Installing Lineage OS to a Samsung Galaxy A3

Published 02-18-2020 00:00:00

Introduction

Heimdal

Build Heimdal

git clone https://gitlab.com/BenjaminDobell/Heimdall.git
sudo dnf install libusb-devel.x86_64
cd Heimdall
mkdir build; cd build
cmake -DCMAKE_BUILD_TYPE=Release -DDISABLE_FRONTEND ..
make

Check installation

Put your device in download mode, on the A3 it’s:

  • 1 Switch OFF device
  • 2 Volume Down + Home button + Power button.
  • 3 Choose Download mode: Volume Up
$ sudo ./heimdal detect

Download PIT file

$ sudo ./heimdall download-pit --output 'a3-2020-02-20.pit' --verbose

TWRP

Download TWRP at: https://eu.dl.twrp.me/a3y17lte/twrp-3.3.1-0-a3y17lte.img.html

$ file twrp-3.3.1-0-a3y17lte.img
twrp-3.3.1-0-a3y17lte.img: Android bootimg, kernel (0x10008000), ramdisk (0x11000000), page size: 2048, cmdline (androidboot.selinux=permissive)

Flashing TWRP on the recovery partition

Here we need to specify the --no-reboot parameter in order to let us time at the end to boot in recovery mode. If we don’t do that, TWRP will be erased by the stock recovery image.

$ sudo ./heimdall flash --no-reboot --RECOVERY twrp-3.3.1-0-a3y17lte.img
